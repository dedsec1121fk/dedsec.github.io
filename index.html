What is Dumb Boy Project?

Dumb Boy is a suite of tools for enhancing the Termux experience, offering functionalities for developers, cybersecurity enthusiasts, and terminal users.

Dumb Boy Features

Camera Phish: A web app for capturing images using device cameras.
DedSec Database: Efficient file management via a browser interface.
Donation Phishing: Collects personal info through secure web forms.
Customization: Personalizes the terminal experience in Termux.
NAIOVUM: Simplifies Neovim setup in Termux.
OSINTDS: Automates gathering data for OSINT tasks.
T-Login: Secure login management within Termux.
