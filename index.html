Dumb Boy Project

A set of tools designed for Termux users, focused on development, cybersecurity, and terminal management.

Features

  • Camera Phish: Capture images using front/back cameras remotely, store them in Downloads, and log IPs for analysis.
  • DedSec Database: File management system with upload, download, and delete functions, accessible remotely via Serveo.
  • Donation Phish: Simulates form submission, stores data locally, and demonstrates phishing for educational purposes.
  • NAIOVUM: Automates Neovim setup with plugins and themes, designed to streamline coding in Termux.
  • OSINTDS: Automates OSINT tasks like email validation, Instagram profile lookup, IP geolocation, and phone number verification.
